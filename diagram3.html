‚è∞ Clock In Flow

1. Employee enters UID at Kiosk
2. System validates employee
3. Check for scheduled shift
4. Validate early/late rules
5. Create timesheet record
6. Send no-show alerts if late

üìù Change Request Flow

1. Employee submits profile change
2. System creates pending request
3. Notification to managers
4. Manager reviews each field
5. Accept/Reject per field
6. Apply changes to profile

üéì Onboarding Flow

1. HR creates employee record
2. Send onboarding link
3. Employee fills personal info
4. Bank & tax details
5. Upload documents
6. HR approves & syncs
